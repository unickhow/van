# VValai

「哇來」 Vite + Vue3 + Typescript for starter template.

## Start
```
npx degit unickhow/vvalai <project_name>

cd <project_name>

pnpm install
```

## Usage
- Development

local serve http://localhost:13845
```
pnpm dev
```
- Build
```
pnpm build
```
preview build result
```
pnpm preview
```
